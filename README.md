# CTF Analysis Toolkit (Python) 

Python based toolkit developed to support analysis tasks commonly encountered in Capture the Flag (CTF) Challenges

Toolkit focuses on automating repetitive analysis steps and applying structured investigation workflows in challenge artifacts

# Purpose 
- Practice analyzing security related artifacts
- Reduce manual effort during CTF challenges
- Reinforce investigation and problem solving skills relevant to entry-level security operations

Developed as a hands-on learning experience in cybersecurity fundamentals 

# Features 
- Decoding encoded data (base64 etc.)
- Parsing and inspecting text based logs/files
- Identifying relevant indicators and extracting data 

Scripts are modular and designed to run depending on challenges 

# What this Toolkit is NOT 
- Not a SIEM
- Not detecting or monitoring platform
- Not intended for production

Designed for learning, experimentation and CTF-style analysis 

# Technologies used 
- Python 3
- Standard Python libraries (argparse, base64, os, sys)

# How it was used 
CTF Challenges and practice labs:
- Automate common decoding and parsing
- Apply repeatable analysis workflows
- Validate findings through scripting

# Future improvements 
- Improved error handling
- More consistent input/output formatting

# Disclaimer
This project is solely educational. Updating and adjusting as I work on more challenges.
